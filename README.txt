Perceptrón Multicapa

- Para poder testear los distintos modelos, utilice el script llamado "MLP_MODELS.ipynb"
- Al tratarse de una ejecución en clúster, los datos los se cargan remotamente desde un repositorio en GitHub
- El fichero "best_model.h5" contiene el modelo que ofrece las mejores prestaciones
- El resto de ficheros recogen la evolución de los MSE de entrenamiento y validación para dicho modelo "óptimo"
